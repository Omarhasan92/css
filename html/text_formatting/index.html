HTML
HyperText Markup Language used to structure content on the web.
CSS
Cascading Style Sheets used for styling HTML elements.
JavaScript
A programming language that adds interactivity to web pages.
DOM
Document Object Model – a tree structure representing HTML elements.
Frontend
The part of a website users interact with, built using HTML, CSS, and JavaScript.
Backend
The server-side logic and database operations powering a website.
API
Application Programming Interface – a way for web services or software to communicate.
HTTP
HyperText Transfer Protocol – the foundation of data exchange on the web.
HTTPS
A secure version of HTTP using encryption (SSL/TLS).
URL
Uniform Resource Locator – the address used to access a web resource.
SEO
Search Engine Optimization – techniques to improve search engine visibility.
Responsive Design
Design that adapts to different screen sizes and devices.
Bootstrap
A CSS framework for building responsive, mobile-first websites quickly.
Flexbox
A CSS layout model for aligning and distributing space within a container.
Grid
A CSS layout system for arranging elements in rows and columns.
Git
A version control system for tracking code changes.
Repository
A storage space for code, often hosted on platforms like GitHub.
Semantic HTML
HTML that uses meaningful tags to improve accessibility and readability.
Form
An HTML element used to collect user input.
Meta Tag
HTML tag in the head that provides metadata like description, author, and viewport settings.
